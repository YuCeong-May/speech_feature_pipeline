# input_audio 数据说明

## 20230829逄JJ积极情绪事件访谈

本目录中的 `20230829逄JJ积极情绪事件访谈.qwen3_forced_align.json` 是使用 Qwen3-ForcedAligner 对
`../../input_audio/20230829逄JJ积极情绪事件访谈.wav` 和对应转录文本
`../../input_audio/20230829逄JJ积极情绪事件访谈.txt` 进行 forced alignment 得到的时间戳结果。

需要注意的是，转录文本最后一段：

```text
嗯
应该没有了
```

在实际音频中基本没有对应的清晰发音。因此 forced aligner 无法为这几个词找到稳定的真实发音区间，
导致末尾若干 token 的时间戳被集中压到同一位置附近。当前对齐结果中可以看到：

```text
嗯 / 应 / 该 / 没 / 有 -> 83.04-83.04
了 -> 86.16-86.16
```

因此，后续计算语速、停顿时间、发音时间、平均音节时长等指标时，不应把这几个末尾 token 当作真实发音片段处理。
建议在计算前进行人工截断或过滤，例如以“喜悦”作为最后一个有效发音词，或过滤掉持续时间为 0 且位于末尾的异常对齐项。
